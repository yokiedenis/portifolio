    Static Vs Dynamic

    Static

    • All content is hard coded
    • Content Remains the same for every viewer
    • Less maintenance costs
    • Limited user interaction
    • Faster to load

    Dynamic

    • Content changes for every single User
    • Not hard coded
    • High maintenance
    • High interactivity
    • Slower to load
